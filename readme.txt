== Description ==

Este plugin aplica um desconto de 5% no valor normal do produto e exibe uma mensagem informando o desconto adicional de 4% para pagamentos via Pix. Adiciona um campo de quantidade ao lado do botão de compra na listagem de produtos.

== Installation ==

1. Envie os arquivos do plugin para o diretório `/wp-content/plugins/woocommerce-pix-discount` ou instale diretamente pelo painel do WordPress.
2. Ative o plugin no menu 'Plugins' do WordPress.
3. Configure as opções do plugin conforme necessário.

== Frequently Asked Questions ==

= Como o plugin aplica os descontos? =
O plugin aplica automaticamente um desconto de 5% ao preço normal do produto e exibe um desconto adicional de 4% para pagamentos via Pix.
